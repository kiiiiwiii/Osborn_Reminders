# 萧逸的温馨提示

一个温馨的桌面提示程序，包含随机出现的温馨话语。

## 使用方法

windows:
1. 下载`萧逸的温馨提示.exe`
2. 双击运行 `萧逸的温馨提示.exe`
3. 系统可能会认为有病毒（当然没有！）各位夫人可以放心run anyway
4. 关闭第一个窗口后会出现多个随机提示窗口
5. 按空格键或关闭任意窗口可退出程序

mac:
1. 下载`萧逸的温馨提示_mac.zip` 
2. 依然是因为安全性的问题，夫人们可能无法打开app，首先打开 系统设置 → 隐私与安全性
3. 向下滚动到 安全性 部分
4. 这里应该会显示关于"萧逸的温馨提示"的阻止信息
5. 点击旁边的 "仍要打开" 按钮
6. 输入你的管理员密码确认
7. 再回到app的位置就可以run anyway啦

## 系统要求

- Windows 7/8/10/11
- 不需要安装 Python
